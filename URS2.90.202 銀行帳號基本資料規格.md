# URS2.90.202 銀行帳號基本資料

## 畫面概述
本畫面提供銀行帳號基本資料的查詢、新增與維護功能，供資金管理課人員管理所有往來銀行帳號資訊。

## 二、操作區

### 【查詢】鈕
1. 查詢成功：
   - {訊息區} 顯示：「查詢成功」
   - 資料依據查詢條件顯示於 {顯示區}

### 【新增】鈕
1. 跳出「銀行帳號基本資料新增」視窗畫面

### 【儲存】鈕

#### 檢核規則：

| 檢核項目 | 檢核條件 | 錯誤訊息 |
|---------|---------|---------|
| A. 勾選項目 | 如未勾選項目 | 「請先勾選項目再執行」 |
| B. 必填欄位 | 如未填寫必填欄位 | 「請填寫必填欄位」 |
| C-i. 資料重複（帳號） | 「銀行代碼」+「銀行帳號(短)」組合已存在 | 「銀行代碼：【XXXXX】，銀行帳號(短)：【XXXXX】；資料已存在」 |
| C-ii. 資料重複（子目） | 「銀存子目」已存在 | 「銀存子目：【XXX】，資料已存在」 |
| D. 同行主調度帳戶 | 該「銀行代碼」的銀行總行代號已有其他帳號被設為「同行主調度帳戶」 | 「已有銀行代碼：【XXXXX】設定為同行主調度帳戶」 |

#### 儲存結果：

| 結果 | 處理 |
|------|------|
| 檢核不通過 | 將檢核不通過訊息顯示於 {訊息區}，顯示「儲存失敗」 |
| 檢核通過 | 資料依據維護資訊儲存，{訊息區} 顯示「儲存成功」，資料依據查詢條件重新顯示於 {顯示區} |

## 五、關聯資料檔彙整

| 資料檔名稱 | 使用欄位 |
|-----------|---------|
| 銀行基本資料檔 | 銀行總行代號、銀行代碼、銀行簡稱 |
| 銀行帳號基本資料檔 | 銀行帳號短碼、銀行帳號長碼、銀行代碼、排列序號、存款類別、幣別類型、銀存子目、帳冊別、外幣帳戶類型、記帳幣、暫收帳戶、FEDI帳戶、公司主調度帳戶、同行主調度帳戶、票匯、活存領息方式、活存計息天數、備註、開戶日期、停用日期、銀行結清日期、公司結清日期、異動人員、異動時間、是否啟用 |
| 銀行幣別基本資料檔 | 幣別 |
| 對照碼檔-存款類別 | 活存、支存、綜存 |
| 對照碼檔-帳冊別 | 不分紅、分紅、利變、OIU |
| 對照碼檔-幣別類型 | 台幣、外幣 |
| 對照碼檔-外幣帳戶類型 | 一般帳戶、外幣保單、OIU |
| 對照碼檔-活存領息方式 | 年領、半年領、季領、月領、無 |
| 對照碼檔-活存計息天數 | 360天、365天 |
